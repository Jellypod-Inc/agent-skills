# Jellypod - Agent Plugins
This repo provides agent plugins (including skills) for building better web applications.

## Installation Guide (Claude Code)
This repository includes a claude plugin marketplace, enabling you to easily install our skills into your repo.

To install this marketplace into Claude Code (and it's associated plugins/skills):
1. Go to Claude Code via `claude`
2. Type `/plugins` and go to the Marketplace tab.
3. Add this repo's full url (https://github.com/Jellypod-Inc/agent-plugins) in the "Add Marketplace".
4. Once the Marketplace has been adding, click 'i' to install any skills.

The official plugin marketplace documentation can be found here: [https://code.claude.com/docs/en/discover-plugins](https://code.claude.com/docs/en/discover-plugins)

## Skills

### Base UI Best Practices
This skill provides an agent optimized version of the Base UI documentation which contains all 25+ components, best practices, and migration callouts from Radix.

The skill is useful when writing new React components, especially when migrating from ShadCN using Radix to ShadCN with Base UI.
